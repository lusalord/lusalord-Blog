1학년 1학기 개인 프로젝트

작성일: 2025-12-16 | 카테고리: 프로젝트

1. 개요

제목 : Pipe and Dungeon
개발 언어 : Unity, C#
개발 인원 : 1인 개발
깃허브 주소 : https://github.com/lusalord/EngineProject

2. 소개

Pipe and Dungeon은 1학년 1학기 개발한 유니티 개인 프로젝트 작품입니다. 저는 여기서
  • 유니티 Input Action을 SO로 변환하여 플레이어의 움직임 제작
  • 인터페이스를 이용하여 아이템 제작
  • HealthSystem과 연동한 체력바 구현
  • Overlab을 이용하여 적의 플레이어 탐지 기능
    • 을 제작하였습니다.

3. 기획 의도

비오는 날에는 파전과 막걸리를 먹어야 한다는 말이 있습니다.
그 말에서 영감을 받아 만든 Pipe and Dungeon은
막걸리를 의인화 시켜 전들과 전투를 하여 적을 물리쳐 적들의 능력을 얻고,
최종적으로 마지막 보스인 빈대떡을 물리치는 게임으로 기획하였습니다.

4. 개발한 기능

Input Action

유니티 Input Action을 Scriptable Object로 변환하여 플레이어의 움직임을 제작하였습니다.
이를 통해 다양한 입력 장치에 대응할 수 있도록 하였습니다.

인터페이스를 이용한 아이템 제작

인터페이스를 활용하여 다양한 아이템을 제작하였습니다.
각 아이템은 공통된 인터페이스를 상속받아 일관된 기능을 가지고 있도록 구현하였습니다.

체력바 구현

HealthSystem과 연동하여 플레이어와 적의 체력바를 구현하였습니다.
체력 변화에 따라 실시간으로 체력바가 변경 되도록 제작하였습니다.

적의 플레이어 탐지 기능

Overlab을 이용하여 적이 플레이어를 탐지하는 기능을 구현하였습니다.
이를 통해 적이 플레이어를 인식하고 반응할 수 있도록 하였습니다.

5. 소감

가장 아쉬운 점은 제출 날 확인을 하지 못하고 맵이 미리 닫혀버리는 버그가 생겼다는 것입니다.
그리고 아트를 자체 제작으로 진행 하였는데 이 과정에서 코드를 짜는 것보다 시간이 더 많이 소요되어 구현하지 못한 기능이 좀 많았던 것 같습니다.
다음 프로젝트에서는 아트 작업 시간을 줄이고, 구현할 기능들을 더 많이 구현할 수 있도록 노력해야겠다는 생각입니다.
