2025년 비공식 게임잼

작성일: 2025-12-17 | 카테고리: 프로젝트

1. 개요

팀명 : FlipFlop
게임 이름 : OverFlow
개발 언어 : Unity, C#
개발 인원 : 5인(개발 4, 아트 1)
깃허브 주소 : https://github.com/CheolYee/OverFlow

2. 소개

OverFlow는 2025년 비공식 게임잼에서 제작한 팀 프로젝트 게임입니다. 저는 여기서
  • 메인 UI 개발
  • 서브 UI 디자인
을 담당하였습니다.

3. 기획 의도

게임잼의 주제는 Limited space와, Running out of space라는 주제였습니다.

이를 바탕으로 저희는 윈도우 xp와 98을 모티브로 삼아,
한정된 메모리 안에서 필요없는 바이러스 파일들을 정리하여 컴퓨터의 메모리를 확보하는 게임을 제작하였습니다.

4. 소감

이번 게임잼을 통해서 내가 짧은 시간 내에 어디까지 개발할 수 있는지와,
팀원들과의 빠른 소통을 통해 협업 능력을 기르는 경험을 할 수 있었습니다.
